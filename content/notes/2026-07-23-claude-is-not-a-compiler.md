**What it is:** exe.dev essay arguing that Claude and coding agents are not compilers, but vertically integrated engineering resources.

**Gist:** The post rejects the idea that Claude is merely a natural-language-to-code compiler. A compiler hides one lower-level decision layer; a coding agent can move across strategy, product, architecture, implementation, tests, adversarial review, and operational failure modes. The concrete exe.dev DNS example shows agents producing divergent full-system implementations, surfacing cross-layer questions, and helping the author build a “scar-tissue” spec that captured the important decisions before the final keeper implementation.

**Newsletter angle:** Strong agents and developer-tools item because it draws a useful boundary between vibe-coding as handing off decisions and vibe-engineering as using agents to accelerate and record cross-layer judgment.
